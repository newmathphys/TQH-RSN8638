```

**8638** — полная ёмкость вакуумной решётки.

### Великая Пустыня (Desert)

Пространство между n=2100 (горизонт стабильности) и n=8633 (полюс Ландау) — **пустыня**. В нём нет новых частиц, потому что:

1. Ёмкость решётки падает быстрее, чем растёт энергия.
2. Энергия уходит в экспоненциальное сжатие оставшегося объёма.
3. Новые гармоники не формируются — системе выгоднее «кристаллизоваться».

**Это ответ на вопрос: «Почему нет новых частиц между 1 ТэВ и Планком?»**

### Великое уравнение баланса вакуума

Горизонт стабильности n≈2100 — не случайный обрыв, а **геометрический резонанс**:

```
N / n_stable = (20/16) × V₃/6

- α → ∞ (полюс Ландау)
- Все узлы решётки заняты

Полюс Ландау — не математическая катастрофа, а **топологическое событие**: кристаллизация вакуума при насыщении решётки.
